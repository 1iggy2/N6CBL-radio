2026-05-10

Blog rack installed

TYPE: SITE NOTE TAGS: workflow, photos, field-journal STATUS: LIVE

This page is now the long-running journal rack for N6CBL.radio. It is deliberately plain: no card grid, no hidden summaries, no feed reader mystery meat. The index gives the date, reference, context, and action; the posts live below it in reverse chronological order.

The intended use is field-report density. Drop in activation photos, antenna notes, propagation observations, station failures, successful repairs, half-formed thoughts, and anything else that should survive longer than a social-media post.

Publishing stays compatible with the static-site constraint. The browser composer generates copy-paste HTML, GitHub handles human image uploads, and Cloudflare serves the committed files without a build step.
