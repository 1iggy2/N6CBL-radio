What these tones do
CTCSS (Continuous Tone-Coded Squelch System, marketed by Motorola as PL — "Private Line") opens a repeater by transmitting a continuous sub-audible tone (67.0–254.1 Hz) underneath the voice audio. The repeater listens for the correct tone and only keys up when it matches. DCS (Digital Code Squelch) does the same job with a slow continuous digital pattern (octal code) instead of an analog tone — better at rejecting interference, but the radio and repeater must agree on the code. The tone or code is sent on transmit; some repeaters also send the tone back on the downlink so receivers can run "tone squelch" and stay silent unless the matching tone is present. Band offsets shown here are common US conventions, not law — always confirm with the local repeater directory or coordinator before programming a memory channel.
