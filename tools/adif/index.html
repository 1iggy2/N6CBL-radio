About ADIF
ADIF (Amateur Data Interchange Format) is the universal logbook exchange format — essentially every popular logger (N1MM, Log4OM, ACLog, WSJT-X, fldigi, hamlog, QRZ, LoTW upload tools) imports and exports it. Each field is tagged like <call:5>N6CBL where the number is the byte length of the value that follows; records are terminated by <eor>. The optional header section before <eoh> carries metadata (program ID, version, character set) that this tool ignores. User-defined fields and any column not in the table above are preserved when round-tripping but hidden from view. Nothing leaves your browser: the file is read with the FileReader API and parsed locally; the export buttons trigger a local download.
