Notes on Cabrillo v3
Cabrillo v3 is the contest log submission format used by virtually every major contest sponsor (CQ, ARRL, WPX, NAQP, RTTY-RU, IARU). The header carries the entry's class and operator metadata; the body is one QSO: line per contact. QSO line format varies by contest — modes have different exchanges (RST + serial, RST + state/section, RST + zone, RST + grid, etc.). This tool emits the generic “RST + serial/exchange” layout: QSO: freq mode date time mycall myrst myexch hiscall hisrst hisexch. Verify with the contest sponsor before submitting — some events (Sweepstakes, Field Day, VHF) use longer exchanges that you may need to edit by hand. For complex exchanges, edit the SENT EXCHANGE field (and the per-QSO STX_STRING in your ADIF source) and add explanatory text to SOAPBOX before submitting.
