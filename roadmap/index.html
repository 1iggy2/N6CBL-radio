Idea exchange protocol how votes work
Stage Required fields Promotion rule
Intake Title, route if known, problem statement, user affected, proposed result. An idea enters the ledger only when it describes a real site or operating problem, not just a cool technology.
Vote Upvote, downvote, defer, and one-sentence rationale. Votes are advisory signal. A low-vote maintenance fix can outrank a popular decorative feature.
Shape Acceptance criteria, data model, privacy risk, mobile risk, and maintenance owner. The feature must have enough definition that a future implementation can be reviewed against objective criteria.
Build Files changed, validation command, rollback path, and final route exposure. A project is build-ready only when it can ship as static-first HTML/CSS/JS or has a documented reason to use the Worker.
Review Post-ship result, defects, usage signal, and next decision. A shipped feature returns to the roadmap if it creates follow-up work or proves a design assumption wrong.
Design language commitments rationale
Commitment Reason Effect on future work
Tables before cards The site is a reference surface. Tables expose comparison, missing data, timestamps, and status better than decorative card grids. Posts, prints, tools, and ideas start as rows.
State is visible Visitors should know whether data is live, stale, manual, generated, planned, or deferred. Every data feature needs generated dates and empty states.
Static-first Performance and durability matter more than editing convenience. Dynamic behavior must earn its operational cost. Workers are used for submissions, proxying, or moderation only when static files cannot do the job.
Permanent navigation The site structure is itself content. Hidden menus would make planned work and obscure utilities harder to discover. New routes must appear in the sidebar or remain non-routable drafts.
No fake polish Gradients, shadows, rounded cards, animations, and vague landing-page copy do not help a radio operator find facts. Complexity goes into metadata, not decoration.
